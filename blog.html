১) প্রশ্নঃ Var, let , Const এর মধ্যে পার্থক্য কি ?

উত্তরঃ Var দিয়ে যদি কোন Veriable ডিক্লার করা হয় তাহলে সেই Veriable স্কোপের বাহিরেও ব্যবহার করা যায় , যা Global Veriable ও বলা যায় , এবংকি সেই Veriable বার বার ডিক্লার করা যায়। যা আমরা ডিক্লার করার আগে ব্যবহার করতে চাইলে undefined দেখায় কিন্তু Error দেয় না। কিন্তু let আর Const এরা একি রকম একবারের বেশি , একি Veriable নাম ব্যবহার করা যায় না, করতে গেলে Error মেরে দেয়। এবং let আর Const এর মধ্যে পার্থক্য হচ্ছে Const ডিক্লার করার পর এর মান পরিবর্তন করা যায় না ।কিন্তু প্রোপার্টির Value পরিবর্তন করা যায় । কিন্তু let একটু স্বাধীন , ডিক্লার করা Veriable এর Value সরাসরি পরিবর্তন করা যায় ।

২) প্রশ্নঃ Array function এবং Regular function এর মধ্যে পার্থক্য কি ?

উত্তরঃ regular function এবং Array function এর মধ্যে পার্থক্য হচ্ছে প্রথম মত this নিয়ে, regular function নিজস্ব this আছে যা প্যারামিটার দিয়ে ধরা যায়। কিন্তু Array function এর নিজস্ব কোন this নেই। যদিও this দিলে কাজ করে তবে তা পুরো Global ধরে নেয়। এবং syntax // Regular Function function regular() { console.log('regularFunction'); return'lol'; } // Array Function const array = ()=> console.log('regularFunction'); ``` Regular function , function দিয়ে শুরু করতে হয় এবং curly bracket এর মধ্যে লিখতে হয় এবং রিটার্ন করতে হলে return keyword ব্যবহার করতে হয় । Array function , Veriable নাম ডিক্লার করে এক্সপ্রেশন দিয়ে শুরু করতে হয় যেখানে ১ টি প্যরামিটার থাকলে প্রথম ব্রাকেট নাহ দিলেও চলে , এবং বিশেষ করে রিটার্ন করতে হলে যদি এক লাইনেই function হয় তাহলে return করা লাগে না এমনেই রিটার্ন করে দেয় । এবং Array function কে চেপ্টা function ও বলা হয় ।

৩) প্রশ্নঃ map ,forEach,filter,find এদের মধ্যে পার্থক্য কি ?

উত্তরঃ map , forEach, filter, find এরা সবাই Array নিয়ে কাজ করে । map হলো Array প্রত্যেকটি Element কে ধরে এবং সব গুলোকে পৃথক ভাবে নিয়ে কাজ করে । এবং সব শেষ ফলাফল রিটার্ন করা যায় । forEach হচ্ছে map এর মত শুধু রিটার্ন করা যায় না। filter হচ্ছে Array প্রত্যেকটি Element কে ধরে এবং সব গুলোকে পৃথক ভাবে **কন্ডিশনের** মাধ্যমে কাজ করে । এবং সব শেষ ফলাফল রিটার্ন করা যায় । find হচ্ছে filter এর মত কিন্তু Array প্রত্যেকটি Element কে ধরে **কন্ডিশনের সাথে মিল রেখে** বাকি গুলোকে Remove করে ফেলে। এবং সব শেষ ফলাফল রিটার্ন করা যায় ।

৪) প্রশ্নঃ `` ট্রাম্পেট স্ট্রিং দিয়ে কি করে এর লাভ কি ?

উত্তরঃ `` ট্রাম্পেট স্ট্রিং এর মজাদার কাজ হচ্ছে ওয়েব পেজ কে ডাইনামিক ভাবে দেখানো যায় ,এবং কোন একটি লেখাকে ভিন্ন ভিন্ন লাইনে দেখাতে গেলে অনেক কোড লিখতে হয় যা ট্রাম্পেট স্ট্রিং দিয়ে অনেক সহজে লিখা ফেলা যায় শুধু ` ` ব্যবহার করে ।
